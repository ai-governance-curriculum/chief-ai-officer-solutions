# Module 101 — Reference Solutions

Reference solutions for [`ai-infra-chief-ai-officer-learning / lessons/mod-101-foundations`](https://github.com/ai-governance-curriculum/chief-ai-officer-learning/tree/main/lessons/mod-101-foundations).

Each exercise has its own subdirectory with a `SOLUTION.md`.
These are *worked answers* — one defensible position per
exercise, with the reasoning shown. **They are not the only
correct answers.** AI governance is a discipline where
multiple structures can be defensible for the same scenario;
the solutions are written to demonstrate rigor, not to be
copied.

## How to use these solutions

1. **Work the exercise first.** Without exception. The point
   of the exercises is the work, not the answer.
2. **Compare against the reference.** Identify where your
   answer diverges. For each divergence, decide whether the
   reference is more defensible or yours is. Defensibility,
   not surface similarity, is the standard.
3. **Read the *reasoning* block at the end of every
   solution.** That is where the real value is. The
   reasoning section names trade-offs the exercise itself did
   not explicitly call out.
