worked synthesis memo at day 90. Same conventions as
the module exercises — one defensible position
per deliverable, with reasoning notes.

plausibly run them**. Many of the decisions could
go differently with equally defensible reasoning.
The reasoning notes name where the solutions could
have gone differently.
